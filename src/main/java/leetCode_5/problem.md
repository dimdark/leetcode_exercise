## Longest Palindromic Substring
---
Description<br/>
Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.

Example:<br/>
```
Input: "babad"
Output: "bab"
```
Note: "aba" is also a valid answer.

Example:<br/>
```
Input: "cbbd"
Output: "bb"
```
